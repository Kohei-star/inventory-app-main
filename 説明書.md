# 在庫管理アプリ 使い方説明書

**対象：全職員**

---

## なぜこのアプリを使うのか

以前は月1回まとめて在庫を入力していたため、その間におむつなどの消耗品が**在庫切れになっても気づけない**という問題が起きました。
「使ったときにすぐ記録する」ことで、リアルタイムに在庫を把握し、在庫切れを防ぐために使います。

---

## 基本ルール（必ずお守りください）

- **使ったらその場で記録する**（後でまとめてはNG）
- **入庫したらその場で記録する**
- 記録するとき必ず**職員名を選ぶ**

---

## アプリの開き方

スマートフォンまたはタブレットのブラウザで施設のURLを開いてください。
ホーム画面に追加しておくとアプリのように使えます。

---

## 画面の見方

### ホーム画面

```
[ 在庫不足アラート ]  ← 在庫が少ない品目がここに出る
[ 検索バー ]
[ すべて｜施設備品｜販売品｜アロマ｜おむつ ]  ← カテゴリで絞り込み
[ No順｜名前順｜在庫数順 ]  ← 並び替え
[ 品目一覧 ]
```

各品目カードに「使用 −」「入庫 ＋」ボタンがあります。

---

## 出庫の記録（使用したとき）

おむつや消耗品を使ったら、**その場で**記録します。

1. ホーム画面で「おむつ」カテゴリを選ぶ
2. 使った品目の **「使用 −」** ボタンを押す
3. 使用した個数を入力する
4. 利用者名を選ぶ（対象の方がいる場合）
5. **職員名を選ぶ**（必須）
6. 「登録」を押す

**ポイント：** 開封したら入力します。開封しているものを1枚使っても入力する必要はありません。

---

## 入庫の記録（納品・補充されたとき）

物品が届いたり、倉庫から補充したときに記録します。

1. ホーム画面で該当の品目を探す
2. **「入庫 ＋」** ボタンを押す
3. 箱数または個数を入力する
   - 入数（1箱あたりの個数）が設定されている品目は「1箱まるごと入庫」ボタンが使えます
4. 「登録」を押す

---

## 在庫不足アラートの見方

ホーム画面の上部に赤いアラートが表示されたら、在庫が最低数を下回っています。

```
在庫の補充をご確認ください（2品目）
  おむつ Mサイズ    残 5個
  プラスチックグローブ  残 3個
```

このアラートが出たら、**すぐに発注または補充の手配**をしてください。

---

## カテゴリ・並び替えの使い方

| ボタン | 意味 |
|--------|------|
| すべて | 全品目を表示 |
| おむつ | おむつ類だけ表示 |
| 施設備品 | 施設で使う備品 |
| 販売品 | ご利用者への販売品 |
| アロマ | アロマ関連品 |
| No順 | 品目番号順（デフォルト） |
| 名前順 | 五十音順 |
| 在庫数順 | 在庫が少ない順 |

---

## Q.A

**Q. 間違えて登録してしまったら？**
舩津にお知らせください。修正対応します。

**Q. 品目が見つからない場合は？**
検索バーに品目名を入力して探してください。それでも見つからない場合は舩津に連絡してください。


---

## まとめ

| やること | タイミング |
|----------|-----------|
| 使用を記録する | **使ったとき、その場で** |
| 入庫を記録する | **届いたとき、その場で** |
| アラートを確認する | **毎日ホーム画面を開いたとき** |

**「後でまとめて」は在庫切れの原因になります。その場で記録することが大切です。**
