# Connect Place 在庫管理アプリ 使用マニュアル

---

## アプリの概要

消耗品・備品の在庫をスマートフォンから手軽に管理できるアプリです。
入庫・使用の記録、棚卸し、レポート確認、品目の追加・編集ができます。

**アクセス方法**
- ブラウザでURLを開く
- または Safari → 共有ボタン →「ホーム画面に追加」でアプリとして使用可能（推奨）

---

## 画面構成

画面下部のナビゲーションバーから各機能に移動します。

| タブ | 機能 |
|------|------|
| 一覧 | 在庫確認・使用/入庫の登録 |
| 棚卸し | 実在庫のカウントと記録 |
| レポート | 使用状況・在庫総額の確認 |
| 品目管理 | 品目の追加・編集・削除 |

---

## 1. 一覧（在庫確認・使用/入庫の登録）

### 在庫を確認する

- アプリを開くと全品目の一覧が表示されます
- 各カードに「品目名」「在庫数」「保管場所」が表示されます
- 在庫が最低在庫数を下回っている品目は**オレンジ色**で警告表示されます

### 品目を検索する

- 上部の検索欄に品目名の一部を入力すると絞り込まれます

### カテゴリで絞り込む

検索欄の下のボタンで絞り込みができます。

- **すべて**：全品目を表示
- **消耗品**：グローブ・テープ類など
- **販売品**：お客様に販売する品目
- **備品**：機器・什器など

### 使用を記録する（使用 −）

1. 該当品目の **「使用 −」** ボタンをタップ
2. 数量を確認（デフォルト：1）
3. **「登録」** をタップ → 在庫数が減ります

> **数量を変更したい場合**：「−」「＋」ボタンで調整するか、数字を直接入力してください

### 入庫を記録する（入庫 ＋）

1. 該当品目の **「入庫 ＋」** ボタンをタップ
2. 入庫した数量を入力
3. **「登録」** をタップ → 在庫数が増えます

---

## 2. 棚卸し

実際の在庫数を数えて、システムの数と照合・修正します。

### 棚卸しの手順

1. 下部ナビの **「棚卸し」** をタップ
2. 全品目が一覧表示されます（システム上の在庫数も表示）
3. 実際に数えた数量を各品目に入力します
   - 「−」「＋」ボタンで調整
   - または数字を直接入力
4. システムと差異がある品目は**オレンジ色の枠**で強調表示されます
5. 全品目の確認が終わったら **「棚卸しを保存する」** をタップ

> 保存すると、実数に合わせて在庫数が自動更新されます。差異の記録も保存されます。

---

## 3. レポート

### 確認できる情報

| 項目 | 内容 |
|------|------|
| 在庫総額 | 全品目の（在庫数 × 仕入単価）の合計 |
| 過去30日 使用回数 | 30日以内に使用登録した合計回数 |
| 使用量の多い品目 | 過去30日間の使用数量ランキング（上位10品目） |

### レポートの見方

- **使用欄**（緑）：その品目が30日間で使われた数量
- **入庫欄**（青）：その品目が30日間で入庫された数量
- **右側の金額**：使用数量 × 仕入単価（消費コストの目安）

## 4. 品目管理

### 新しい品目を追加する

**方法①**：一覧画面の一番下の **「＋ 品目を追加する」** をタップ

**方法②**：品目管理タブ → 右上の **「＋ 追加」** をタップ

**入力項目**

| 項目 | 説明 |
|------|------|
| 品目名 ※必須 | 品目の正式名称 |
| 種別 | 消耗品 / 販売品 / 備品 から選択 |
| 現在の在庫数 | 登録時点の在庫数 |
| 最低在庫数 | これを下回ると警告アラートが出ます |
| 購入単位 | 例：1箱、1ロール |
| 仕入単価（円） | レポートの在庫総額・消費コスト計算に使用 |
| 仕入先 | 発注先の業者名 |
| 保管場所 | 例：会社1F倉庫 |
| 販売単価（円） | 種別が「販売品」の場合のみ表示 |

入力後、**「品目を登録する」** をタップして完了です。

---

### 既存の品目を編集する

1. 下部ナビの **「品目管理」** をタップ
2. 編集したい品目の **「編集」** ボタンをタップ
3. 変更したい項目を修正
4. **「変更を保存する」** をタップ

> 在庫数の修正は棚卸し機能からも行えます。

---

### 品目を削除する

1. 品目管理 → 削除したい品目の **「編集」** をタップ
2. 画面下部の **「この品目を削除する」** をタップ
3. 確認ダイアログで **「OK」** を選択

> ⚠️ 削除すると元に戻せません。過去の取引履歴にも影響する場合があります。

---

## よくある操作

### 入庫したのに在庫が反映されていない

→ 「入庫 ＋」で登録後、一覧画面をスクロールして更新されているか確認してください。画面をリロード（再読み込み）すると反映されます。

### 棚卸しで数量を間違えた

→ 保存前であれば数量を再入力できます。保存後に修正する場合は、再度棚卸し画面から正しい数量を入力して保存してください。

### 品目の順番を変えたい

→ 現在は登録順（No.順）で表示されています。検索・カテゴリフィルタで目的の品目を素早く見つけてください。

---

## 注意事項

- インターネット接続が必要です
- 複数人で同時に操作した場合、画面を再読み込みすると最新の在庫数が表示されます
- データはクラウド（Supabase）に保存されています
